# Drizzle ORM

Memori auto-detects a Drizzle `db` instance and routes SQL to the correct dialect automatically. Pass the `drizzle(...)` handle directly — no extra configuration needed.

## Install

Install Drizzle and your underlying database driver:

<CodeGroup title="Install Drizzle">

```bash {{ title: 'SQLite (better-sqlite3)' }}
npm install drizzle-orm better-sqlite3
npm install --save-dev @types/better-sqlite3 drizzle-kit
```

```bash {{ title: 'PostgreSQL (pg)' }}
npm install drizzle-orm pg
npm install --save-dev @types/pg drizzle-kit
```

```bash {{ title: 'MySQL (mysql2)' }}
npm install drizzle-orm mysql2
npm install --save-dev drizzle-kit
```

</CodeGroup>

## Quick Start

<CodeGroup title="Drizzle Connection">

```typescript {{ title: 'SQLite' }}
import 'dotenv/config';
import Database from 'better-sqlite3';
import { drizzle } from 'drizzle-orm/better-sqlite3';
import { OpenAI } from 'openai';
import { Memori } from '@memorilabs/memori';

const client = new OpenAI();
const sqlite = new Database('memori.db');
const db = drizzle(sqlite);

const mem = new Memori({ conn: db }).llm.register(client);
mem.attribution('user-123', 'my-app');

if (!mem.config.storage) {
  throw new Error('Storage not initialized');
}

await mem.config.storage.build();

const response = await client.chat.completions.create({
  model: 'gpt-4o-mini',
  messages: [{ role: 'user', content: 'My favorite color is blue.' }],
});
console.log(`AI: ${response.choices[0]?.message?.content}`);

await mem.augmentation.wait();
await mem.config.storage.close();
```

```typescript {{ title: 'PostgreSQL' }}
import 'dotenv/config';
import pg from 'pg';
import { drizzle } from 'drizzle-orm/node-postgres';
import { OpenAI } from 'openai';
import { Memori } from '@memorilabs/memori';

const client = new OpenAI();
const pool = new pg.Pool({ connectionString: process.env.DATABASE_CONNECTION_STRING });
const db = drizzle(pool);

const mem = new Memori({ conn: db }).llm.register(client);
mem.attribution('user-123', 'my-app');

if (!mem.config.storage) {
  throw new Error('Storage not initialized');
}

await mem.config.storage.build();

const response = await client.chat.completions.create({
  model: 'gpt-4o-mini',
  messages: [{ role: 'user', content: 'My favorite color is blue.' }],
});
console.log(`AI: ${response.choices[0]?.message?.content}`);

await mem.augmentation.wait();
await mem.config.storage.close();
await pool.end();
```

```typescript {{ title: 'MySQL' }}
import 'dotenv/config';
import mysql from 'mysql2/promise';
import { drizzle } from 'drizzle-orm/mysql2';
import { OpenAI } from 'openai';
import { Memori } from '@memorilabs/memori';

const client = new OpenAI();
const pool = mysql.createPool({ uri: process.env.DATABASE_CONNECTION_STRING });
const db = drizzle(pool);

const mem = new Memori({ conn: db }).llm.register(client);
mem.attribution('user-123', 'my-app');

if (!mem.config.storage) {
  throw new Error('Storage not initialized');
}

await mem.config.storage.build();

const response = await client.chat.completions.create({
  model: 'gpt-4o-mini',
  messages: [{ role: 'user', content: 'My favorite color is blue.' }],
});
console.log(`AI: ${response.choices[0]?.message?.content}`);

await mem.augmentation.wait();
await mem.config.storage.close();
await pool.end();
```

## Supported Dialects

| Drizzle Import                       | Dialect Detected |
| ------------------------------------ | ---------------- |
| `drizzle-orm/better-sqlite3`         | SQLite           |
| `drizzle-orm/node-postgres`          | PostgreSQL       |
| `drizzle-orm/mysql2`                 | MySQL            |

## Notes

- Pass the `drizzle(...)` instance, not the underlying driver directly.
- Memori manages its own raw SQL tables alongside your Drizzle schema — no conflicts.
- Drizzle's connection lifecycle (pool management) is your responsibility; call `pool.end()` after `mem.config.storage.close()`.
